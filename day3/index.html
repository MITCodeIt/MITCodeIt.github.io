Example Project

Starter Project
Hint Sheet

A. Change Color and Reset

  1. Make the color sprite broadcast a message.
  2. Make the nature sprite change colors when it receives the message.
  3. Make the reset sprite return images to their original state.
  4. Make the nature sprite clear graphic effects when it receives that message.

B. Ghost, Pixelate, Mosaic, and Fisheye Effects

  1. Program the ghost, pixelate, mosaic, and fisheye sprites to broadcast messages.
  2. Program the nature sprite to change ghost, pixelate, mosaic, and fisheye effects when it receives those messages.

C. Choose Your Image

  1. Select another image to display in your program.
  2. If you choose a costume from the people or things sprites, copy the code from the nature sprite.
  3. If you upload your own image, edit the image so it fits in the image slot on the stage.

Add-ons

  • Enable the user to scroll through different images using a keypress event.
  • Create your own unique filter.
  • Add a surprise feature to your program.
  • Add sound to your project.
  • Program a dropdown menu to give the user a chance to pick the image category displayed.
  • Make the images respond to specific effects.

Example Project

Starter Project
Hint Sheet

A. Draw with the Pencil

  1. Program the pencil to follow the mousepointer.
  2. Make the pencil draw.
  3. Clear the canvas when the program starts.

B. Draw when Mouse Clicked

  1. Add an "if-else" block.
  2. Make the "if" block check if the mouse button is pressed.
  3. If the mouse button is pressed, make the pencil draw.
  4. If the mouse button is not pressed, make the pencil stop drawing.

C. Questions and Answers

  1. Program the monkey sprite to ask player 1 for a drawing.
  2. Program the game so that player 2 can guess the drawing.
  3. Create a variable to store the "right answer."
  4. Program a winning condition if player 2 answers the drawing correctly.

Add-Ons

  • Allow the user to change the pen color (use broadcasting).
  • Allow the user to change the pen size.
  • Make the monkey do a little dance and introduce the game.
  • Make it impossible to draw outside the white spaces.
  • Add sound to your game.
