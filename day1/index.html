Dance Party

Example Project

Starter Project
Hint Sheet

A. Make Cassy Dance in One Spot

  1. Make Cassy look like she is dancing.
  2. Make Cassy keep dancing forever.
  3. Control the speed of Cassy's dance.

B. Make Cassy Move

  1. Use the "glide" block to make Cassy move.
  2. Make the dance start when the green flag is clicked.

C. Add Dance Music for Cassy

  1. Select the stage.
  2. Choose a music loop for Cassy to dance to.
  3. Use the “play sound until done” block to make the music loop play.
  4. Make the music loop play forever.
  5. Make the music loop start playing when the green flag is clicked.

Add-ons

  • Add friends to the dance party
  • Add and change the backdrop
  • Make Cassy do the twist (hint: use the "change whirl effect by" block)
  • Make Cassy tell a story (hint: use "say" blocks)
  • Use different keys to control the music being played
  • Use different keys to control Cassy's dance moves
