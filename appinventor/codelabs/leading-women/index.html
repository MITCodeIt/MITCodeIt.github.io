In this codelab, you will create an app that plays speeches by Michelle Obama and Hillary Clinton.

What you'll learn

  • Use Arrangement components to organize other components like buttons and labels on the screen.
  • Play media using the Player component.
  • Use if-else blocks to control what your app does in different situations.
  • Check a component's properties using code.

Conditional statements using if-else blocks

Conditional statements are fundamental to programming. They allow the program to make decisions based on the current state of the program. Almost all "if" statements check the values of component properties. In this case, the Player component's IsPlaying property is checked, and the program chooses an action based on whether the Player is playing or not.
