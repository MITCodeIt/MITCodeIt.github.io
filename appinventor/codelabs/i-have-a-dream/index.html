Go to Blocks View. Let's add the ability to pause the speech if it is already playing. To do this, we will use an if-else block.

First, drag a when MLKButton.Click block. Then, drag an if block from the Control drawer. Click the blue modifier icon, and drag an else block into the picture of the if block. This modifies your if block to have two branches.

Conditional statements are fundamental to programming. They allow the program to make decisions based on the current state of the program. Almost all "if" statements check the values of component properties or values. In this case, the Player component's IsPlaying property is checked, and the program chooses an action based on whether the Player is playing or not.
