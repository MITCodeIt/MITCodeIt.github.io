Example Project

Starter Project
Hint Sheet

A. Fill the Stage With Color

  1. Use the "go to" block to make the sprite go to a specific spot.
  2. Make the sprite go to a random spot. (hint: The x-coordinates of the canvas go from -240 to 240, and the y-coordinates go from -180 to 180.)
  3. Make the sprite leave a stamp of itself.
  4. Make the sprite repeat going to a random spot and stamping itself.
  5. Start with a clear screen when the green flag is clicked.

B. Color Me Musical

  1. Make the X sprite go to the mouse.
  2. Make the X sprite follow the mouse FOREVER.
  3. Add an "if" block.
  4. Make the "if" block say "if touching color."
  5. Make the X sprite play a note if it is touching green.
  6. Copy the if statement, then change it to make each color in your project play a different note.
  7. Add a "when green flag is clicked" event to start the program.

Add-ons

  • Change instruments and tempo in your musical art piece when the yellow X sprite touches different colors.
  • Make your project play chords, not just individual notes.
  • Add a surprise color to your art piece that plays an unexpected sound when touched.
  • Add a graphic effect that makes the yellow X sprite leave a color-changing trail across the screen. (hint: use "create clone of myself" and "when I start as a clone" blocks)
  • Create another yellow X sprite that moves around your art piece and plays random notes.
  • Make the yellow X sprite change colors, depending on which color it is touching.

Example Project

Use the arrow keys to point the fish in a specific direction, then use the space bar to launch the fish. Get to the finish line!

Starter Project
Hint Sheet

A. Launch a Sprite

  1. Select a Player 1 sprite and create an event that will happen when the spacebar is pressed.
  2. Add a "move" block to make the sprite move forward.
  3. Add a "repeat until" block to the stack.
  4. Add a "touching Finishline" block as a condition.
  5. Set a starting position for your sprite with a "go-to" block.

B. Turn the Sprite Left and Right

  1. Select the player 1 sprite and create events for the left and right arrow keys.
  2. Add two "turn" blocks to make the sprite turn left and right.
  3. Make the player 1 sprite turn smoothly by adding a "repeat until" block, a "not" block, and a "key pressed" block to each "turn" stack.

C. Spawning Enemies

  1. Add a sprite and resize it.
  2. Make the new sprite bounce around the stage.
  3. Make the enemy sprite clone itself.
  4. Make each enemy sprite clone start off pointing in a random direction.
  5. Add "hide" and "show" blocks. (hint: Hide the original sprite and show each clone upon creation.)

D. Keeping Score

  1. Create a "score" variable for the player sprite.
  2. Use an if statement to add to the "score" variable when the player reaches the finish line.
  3. Add a "set score to 0" block when the player clicks on the green flag.
  4. Create an enemy clone every time a point is scored.

Add-ons

  • Program a sound to play if the player touches the finish line.
  • Add a second player to the game.
  • Challenge a friend by having them control the enemy.
  • Create a sprite that, when touched, gives the player extra points.
  • Program the finish line to get smaller as the player scores more points.
  • Program the backdrop to change every time the player scores 5 points.
