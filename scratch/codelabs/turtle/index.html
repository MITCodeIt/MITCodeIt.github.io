Functions

A function is a procedure or a routine. A function is usually associated with a set of instructions. When you call a function, the set of instructions are run.


# This is how you define a function called move_tina in Python.
# It takes an x-coordinate and a y-coordinate and moves tina to that location.
def move_tina(x, y):
  tina.penup()
  tina.goto(x, y)
  tina.pendown()

# This is how you call the move_tina function.
# Tina should move to coordinate (100, 50).
move_tina(100, 50)
