In this codelab, you'll make a game where the player guides a fish to swim between pairs of pipes.

What you'll learn

  • Use the space bar to change the speed of the fish.
  • Use variables to track the fish's score.
  • Use loops to continuously generating pipes until the game ends.

Open the starter project and click Remix.

First, we are going to program the fish's starting position. We will be using two variables: Score and Speed.
