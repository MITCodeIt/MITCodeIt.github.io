In this codelab, you'll learn how to create a launcher game and keep score.

Use the arrow keys to point the fish in a specific direction, then use the space bar to launch the fish. Get to the finish line!

What you'll learn

  • Review loops.
  • Practice using if/else conditional blocks.
  • Practice using variables.
  • Learn about cloning.

Open the starter project and click Remix.

First, we are going to get a sprite moving across the screen until it reaches the finish line.

Useful blocks:

Steps

  1. Select a sprite from the sprite library to be player 1.
  2. Make the sprite move continuously across the screen.
  3. Have the sprite stop once it reaches the finish line.
  4. Set a starting position for your sprite. Your sprite should also return here after touching the finish line.

Next, we are going to control player 1's motion.

Useful block:

Steps

  1. Have your sprite turn as you press the left and right arrow keys.
  2. If necessary, smooth out your sprite's turning.

Launcher Game Checkoff #1

To receive this checkoff, you must answer the following questions:

  • How does the sprite keep moving after you launch it?
  • How does the sprite know to stop at the finish line?
  • How could you speed up or slow down the turning of your sprite?

Now, we are going to add enemies and have them spawn after each win.

Useful blocks:

Steps

  1. Add another sprite as an enemy.
  2. Make the enemy sprite bounce around the stage
  3. Make the enemy sprite clone itself whenever player 1 reaches the finish line.
  4. Have each enemy sprite clone start off pointing in a random direction.

Launcher Game Checkoff #2

To recieve this checkoff, you must answer the following questions:

  • How do the enemies move without flying off the screen?
  • How does the game know when to add new enemies?

Now, we are going to add enemies and have them spawn after each win.

Steps

  1. Create a "score" variable for the player sprite. Write your choice for the name of your variable where it says "name here".
       
  2. Use a conditional to add to the "score" variable when the player reaches the finish line.
  3. Have the score reset to 0 when a person restarts the game.

Launcher Game Checkoff #3

To recieve this checkoff, you must answer the following questions:

  • How does the game know when to increase score?
  • How do you reset the score at the start of the game?
 If you finish early, implement one or more of the following features for a bonus checkoff.

Launcher Game Bonus Add Ons

Implement one or more of the following, and show it off to your mentor!

  • Program a sound to play if the player touches the finish line.
  • Add a second player to the game.
  • Challenge a friend by having them control the enemy.
  • Create a sprite that, when touched, gives the player extra points.
  • Program the finish line to get smaller as the player scores more points.
  • Program the backdrop to change every time the player scores 5 points.
