In this minilab, you'll complete the code to be able to personalize the features of the face on the stage.

When pressing the buttons below the face, they should alter the respective part of the face by switching to the next costume. For example, when clicking the "face" button, the color of the face should change. If the button is a sprite and the face is a sprite, how can we communicate between the button sprite and the face sprite to indicate when it's clicked?

What you'll learn

  • Review the use of conditional statements.
  • Use broadcasting.

Open the starter project and click Remix.

 When you're finished, be sure to save and share your project, and then add it to the studio here .
