In this minilab, you'll practice your broadcasting skills.

Complete the starter code in order to make the Scratch Cat see a morphed image of him/herself in the mirror, like in a Funhouse. You'll need to use broadcasting to help the reflection of the cat know when to warp itself. Program it such that each dot has a different effect.

What you'll learn

  • Review the use of conditional statements and loops.
  • Explore the possibilities of graphic effects in the Looks section of the code.
  • Use cloning.

Important things before you begin

  • You will need to edit both the Scratch Cat's code and it's Reflection.
  • You will want to clear all graphic effects before adding new ones on the Reflection sprite.
  • Each button color that the Scratch Cat touches should let the Reflection know how it will be warped.
  • Make sure that your code is always checking whether the Scratch Cat is touching a button.
  • Try to experiment with the looks effects.

Open the starter project and click Remix.

 When you're finished, be sure to save and share your project, and then add it to your class studio.
